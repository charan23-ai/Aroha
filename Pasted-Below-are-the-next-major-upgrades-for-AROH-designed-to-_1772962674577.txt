# 1. Minimalist Mood Scale (No Emojis)

This replaces sliders and emojis with a **clean cognitive mood spectrum**.

## `/components/MoodScale.tsx`

```tsx
"use client"

const moods = [
  "Very Low",
  "Low",
  "Heavy",
  "Neutral",
  "Okay",
  "Lighter",
  "Good"
]

export default function MoodScale({ mood, setMood }) {

  return (

    <div className="w-full">

      <h2 className="text-lg font-medium mb-6">
        How are you feeling right now?
      </h2>

      <div className="flex justify-between text-xs text-slate-500 mb-2">
        {moods.map((label, i) => (
          <span key={i} className="w-[14%] text-center">
            {label}
          </span>
        ))}
      </div>

      <div className="relative flex items-center">

        <div className="absolute w-full h-[2px] bg-slate-300"/>

        {moods.map((_, i) => (

          <button
            key={i}
            onClick={() => setMood(i)}
            className={`relative w-4 h-4 rounded-full border 
            ${mood === i ? "bg-black border-black scale-125" : "bg-white border-slate-400"}
            transition`}
          />

        ))}

      </div>

    </div>
  )
}
```

Design philosophy:

• neutral language
• avoids emotional labeling pressure
• adolescent-friendly
• clinical but gentle

# 5. AI-Powered Emotional Analysis Engine

This engine detects patterns in:

• mood logs
• menstrual cycle
• activities
• sleep
• journaling

---

## `/lib/emotionAI.ts`

```ts
export function analyzeEmotionalState(data){

let insights=[]

const avgMood=data.mood.reduce((a,b)=>a+b)/data.mood.length

if(avgMood<3){

insights.push(
"Your mood has been consistently low this week. Gentle grounding activities may help restore emotional balance."
)

}

if(data.sleep<6){

insights.push(
"Sleep patterns appear linked to lower mood entries."
)

}

if(data.activities>=3){

insights.push(
"You tend to feel lighter after completing reflection exercises."
)

}

return insights

}
```
